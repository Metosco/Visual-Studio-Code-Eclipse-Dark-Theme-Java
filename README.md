# Eclipse Dark Java

A Java-focused dark color theme for Visual Studio Code, inspired by the Eclipse IDE Dark theme.

## Features

- Dark UI optimized for Java development
- Semantic highlighting enabled for accurate Java token coloring
- Carefully tuned colors for keywords, types, strings, comments, and annotations

## Installation

### From VS Code Marketplace

1. Open **Extensions** sidebar (`Ctrl+Shift+X`)
2. Search for `Eclipse Dark Java`
3. Click **Install**
4. Open **Preferences: Color Theme** and choose **Eclipse Dark Java**

### From VSIX

1. Download the `.vsix` from [Releases](https://github.com/Metosco/Visual-Studio-Code-Eclipse-Dark-Theme-Java/releases)
2. Run `code --install-extension eclipse-dark-java-theme-0.1.0.vsix`
